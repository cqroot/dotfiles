# Keith Nvim

Keith Chu's Nvim Configuration.

## Installation

### Linux

```bash
git clone https://github.com/cqroot/keithnvim ~/.config/nvim

cd ~/.config/nvim
make bootstrap
```

### Windows (Git Bash)

```bash
git clone https://github.com/cqroot/keithnvim "${LOCALAPPDATA}/nvim"

cd "${LOCALAPPDATA}/nvim"
make bootstrap
```

## Check Dependencies

```bash
make check
```
